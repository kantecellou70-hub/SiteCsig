Accessibilité

Nous nous engageons à rendre notre site web accessible à tous les utilisateurs, y compris ceux ayant des besoins spéciaux.

Notre engagement

Le CSIG s'engage à fournir un site web accessible à tous les utilisateurs, conformément aux standards internationaux d'accessibilité web (WCAG 2.1). Nous travaillons continuellement à améliorer l'accessibilité de notre site.

Navigation au clavier

Notre site est entièrement navigable au clavier. Utilisez la touche Tab pour naviguer entre les éléments et Entrée pour activer les liens et boutons. Les raccourcis clavier suivants sont disponibles :

  • Alt + H : Retour à l'accueil
  • Alt + M : Menu principal
  • Alt + C : Page de contact
  • Alt + S : Barre de recherche

Contraste et lisibilité

Nous avons optimisé le contraste des couleurs pour assurer une excellente lisibilité. Tous nos textes respectent un ratio de contraste minimum de 4.5:1, conformément aux recommandations WCAG.

Images et médias

Toutes nos images contiennent des textes alternatifs descriptifs. Les vidéos incluent des sous-titres et des transcriptions. Les éléments décoratifs sont marqués comme tels pour les lecteurs d'écran.

Typographie

Nous utilisons des polices lisibles et des tailles de texte appropriées. Les utilisateurs peuvent ajuster la taille du texte via les paramètres de leur navigateur (Ctrl + "+" pour agrandir, Ctrl + "-" pour réduire).

Responsive design

Notre site s'adapte automatiquement à tous les écrans et appareils. Il est optimisé pour les ordinateurs de bureau, tablettes et smartphones, garantissant une expérience utilisateur cohérente sur tous les supports.

Technologies d'assistance

Notre site est compatible avec les principales technologies d'assistance :

  • Lecteurs d'écran (JAWS, NVDA, VoiceOver)
  • Logiciels de reconnaissance vocale
  • Logiciels de grossissement d'écran
  • Dispositifs de navigation alternative

Limitations connues

Bien que nous nous efforcions de maintenir un niveau d'accessibilité élevé, certaines limitations peuvent exister dans certaines sections du site. Nous travaillons activement à les résoudre.

Signaler un problème

Si vous rencontrez des difficultés d'accessibilité sur notre site, n'hésitez pas à nous contacter. Votre retour nous aide à améliorer l'expérience de tous les utilisateurs.
